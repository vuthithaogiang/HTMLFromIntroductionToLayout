CSS là gì?

CSS dùng để định dạng bố cục trang web. Từ C (Cascading) strong CSS nghĩa là áp dụng cho phần tử gốc rễ cũng sẽ áp dụng cho tất cả các phần tử con trong phần tử gốc.

Với CSS, có thể kiểm soát màu sắc, phông chữ, kích thước của văn bản, khoảng cách giữa các phần tử, cách các phần được định vị và trình bày, hình nền hoặc mầu nền nào sẽ được sử dụng.

CSS cung cấp các cách hiển thị khác nhau cũng như nhiều hơn thế nữa.
